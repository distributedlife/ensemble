# Setup

## OSX

1. install rbenv
2. install version of ruby listed in .ruby-version
3. `gem install bundler`
4. `bundle install`
5. install nodenv
6. install version of node listed in .node-version
7. `brew install mongo`
7. `npm i`
8. `cd node_modules && ln -nsf ../src && cd..`
8. `gulp`

9. `npm i jshint -g`
